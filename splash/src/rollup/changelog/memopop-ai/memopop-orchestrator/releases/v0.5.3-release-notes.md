# Release v0.5.3 — MemoPop Interactive CLI

**Investment Memo Orchestrator** — Multi-agent system for generating professional investment memos using LangGraph and Claude.

This release introduces **MemoPop** — an interactive terminal application that makes the entire system accessible without memorizing commands, flags, or file paths. Type `memopop` and follow the prompts.

---

## Getting Started

### Install

```bash
# Clone and set up (first time)
git clone https://github.com/lossless-group/investment-memo-orchestrator.git
cd investment-memo-orchestrator

# Create venv and install
uv venv --python python3.11
source .venv/bin/activate
uv pip install -e .

# Configure API keys
cp .env.example .env
# Edit .env with your ANTHROPIC_API_KEY (required)
# and PERPLEXITY_API_KEY, TAVILY_API_KEY (recommended)
```

### Launch

```bash
# Activate your venv, then:
memopop
```

That's it. The app discovers your firms, deals, and versions automatically.

## What MemoPop Does

MemoPop is a guided wrapper around the 15+ CLI commands in the system. It auto-discovers everything and presents contextual options:

### 📝 Generate a New Memo

```
? Select firm:
  ❯ Humain Ventures (3 deals)
    Hypernova Capital (5 deals)

? Select deal:
  ❯ Metabologic (latest: v0.2.4 · Seed · deck ✓ · dataroom ✓)
    ProfileHealth (latest: v0.1.0 · Series A · deck ✓)

? Metabologic has existing output at v0.2.4. What would you like to do?
  ❯ Generate fresh (clean slate)
    Resume from v0.2.4
    Start at a specific version number

? Confirm settings:
  ┌─────────────────────────────────┐
  │ Company:  Metabologic           │
  │ Firm:     Humain Ventures       │
  │ Type:     Direct Investment     │
  │ Mode:     Prospective (consider)│
  │ Deck:     ✓ Found               │
  │ Dataroom: ✓ Found               │
  │ Strategy: Fresh start           │
  └─────────────────────────────────┘

? Start generation? (Y/n)
```

The pipeline runs with live output, then offers post-run actions (export, one-pager, improve sections).

### 📤 Export an Existing Memo

Select firm → deal → version → format (HTML+PDF / Word / One-pager / All) → mode (Light / Dark / Both). MemoPop resolves all file paths automatically.

### 🔧 Improve a Specific Section

Select firm → deal → version → section (listed from `2-sections/`). Calls `improve-section.py` with Perplexity Sonar Pro for real-time research enrichment.

### 🔄 Integrate Content from Versions

Scaffolded flows for human-in-the-loop curation:
- **Source review**: See all sources from the source catalog, with section counts
- **Competitive landscape**: Merges competitors across ALL versions with dedup, shows summary table with AI classifications and version history
- **Table proposals** and **Syndicate investigation**: Placeholders for next release

### 📊 Run a Specific Agent

Categorized menu of individual agents (assembly, tables, one-pager, citation spacing, exports) that can be run on any existing output directory.

## Auto-Discovery

MemoPop doesn't require you to know file paths. It scans the `io/` directory and discovers:

| What | How |
|------|-----|
| **Firms** | Directories in `io/` with a `deals/` subfolder |
| **Deals** | Directories with a `{DealName}.json` config file |
| **Versions** | Output directories matching `{Deal}-v*` pattern |
| **Metadata** | Stage, deck/dataroom status, latest version date, curations |
| **Curations** | Detects `_source-curation.json`, `_competitive-curation.json`, etc. |

## Technical Details

### Architecture

```
cli/terminal_app/
├── __init__.py
└── py_rich/              # Rich framework implementation
    ├── __init__.py
    ├── app.py            # Main entry point, menu loop, all flows
    ├── discovery.py      # Auto-detect firms, deals, versions
    └── theme.py          # Color palette, Rich + questionary styling
```

The `py_rich/` directory isolates the Rich-based implementation. A future `astro/` sibling could provide a Node.js alternative with the same flows.

### Dependencies

- **Rich** (already installed) — Panels, tables, rules, themed console output
- **questionary** (new in v0.5.3) — Interactive select menus, confirmations, text prompts
- **prompt-toolkit** (transitive via questionary) — Terminal input handling

### Entry Point Registration

`memopop` is registered as a console script in `pyproject.toml`:

```toml
[project.scripts]
memopop = "cli.terminal_app.py_rich.app:main"
```

After `uv pip install -e .`, the `memopop` binary is available at `.venv/bin/memopop`. With the venv activated, it's on your PATH.

### Color Theme

Matches the Humain Ventures palette (configurable):
- Accent: `#51A084` (Lochinvar green)
- Primary: `#090E18` (Bunker dark)
- Muted: `#656F84` (Blue Bayoux)
