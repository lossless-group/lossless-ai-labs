## what landed

- Astro 6 + Tailwind v4 + Vercel-static + Astro Fonts API (Inter + IBM Plex Mono).
- `theme.css` with three-mode contract (light filled, dark/vibrant TBD placeholders).
- `DESIGN.md` (Stitch-spec format) with values extracted verbatim from trychroma.com production CSS.
- `draft-palette__Chroma.json` mirror of the same tokens.
- `public/brand/chroma-wordmark.svg` downloaded from upstream.
- Routes:
  - `/` — landing page
  - `/scroll/` — thumb-gallery index
  - `/scroll/pitch/` — one stub scroll deck using `PageAsDeckWrapper`
  - `/play/` — empty-state thumb gallery (runtime deferred)
- `corpus/` directory git-ignored, README explaining role.
- `context-v/` directory with conventional subdirs and stubs.

## what's deliberately deferred

- `/play/` runtime — lifted from calmstorm-decks in a later pass.
- Variant pages (`/drafts/{slug}/{slug}-vN`) — calmstorm pattern, kept available but not authored.
- Markdown rendering components — removed from v0.1 (no narratives yet); copy back when content authoring begins.
- Auth gating — open preview for v0.1.
- `dark` and `vibrant` mode palette values — separate brand-iteration session.
